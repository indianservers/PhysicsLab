## 3. Physics Risks Eliminated

| Risk | Before | After |
|---|---|---|
| Fake pressure graph values | `pressure = 101325 + mass * gravity * 2` | Removed. No pressure graph is available without a registered pressure model. |
| Fake voltage graph values | `voltage = abs(charge) * 5` | Removed. Voltage is not graphed from Matter objects. |
| Fake current graph values | `current = abs(charge) * 0.5` | Removed. Current is not graphed from Matter objects. |
| Fake intensity graph values | Distance-based arbitrary intensity | Removed. Intensity is unavailable unless a future optical/acoustic model supplies it. |
| Fake wavelength graph values | Velocity-derived length | Removed. Wavelength is unavailable unless a wave/optics model supplies it. |
| Fake frequency graph values | Speed treated as frequency | Removed. Frequency is unavailable from Matter snapshots. |
| Graphs without unit provenance | Axis labels used local hard-coded helper | Axis labels now use measurement adapter units. |
| Export of misleading measurements | CSV/JSON exported every graph key equally | Export serializes registered measurements and marks unavailable values. |

## 4. Graph Issues Fixed

- Graph selectable variables now come from `graphableVariables` in `measurementAdapters.ts`.
- Every graph trace can carry:
  - source type,
  - scientific confidence,
  - assumptions.
- Graph UI shows source type and confidence for traces.
- CSV export includes metadata rows:
  - Generated By,
  - Model Class,
  - Trust Level,
  - Assumptions,
  - Formula Used.
- JSON export wraps data in a metadata payload.
- Observation auto rows now use the active trace and adapter unit instead of always comparing `point.y` to `point.x` in meters.

## Before vs After

Before:

- Mechanical snapshots fabricated unrelated physics quantities.
- Graph/export system treated placeholder values as real data.
- Units were mostly display strings, not a registry.
- Experiments did not surface model class or trust level.
- Lab reports lacked assumptions and trust metadata.
- Physics tests were absent.

After:

- Fake graph measurements are removed.
- Registered measurement adapters gate graphable quantities.
- Graphs display source type and scientific confidence.
- CSV/JSON exports include safety metadata and unavailable measurements are not fabricated.
- Unit registry, converters, dimensional validation, constants, and pure calculators exist.
- Experiments display model class, trust level, assumptions, limitations, ranges, and failure conditions.
- Physics validation tests pass.
